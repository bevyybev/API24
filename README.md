# API24
>Build Simple CRUD API Using MS Azure & Heroku

## Downloads

- Postman
- Heoku
- Homebrew

## Get Started
> Download files

Install dependancies & run Locally

#### Express

`npm install express`
#### Nodeman

`npm install -g nodemon`

`npm start`

#### Postman
1. Download Postman
2. Create account
3. Create new workbook
4. Test request
   - e.g. Get http://localhost8080
   


####  Push to Heroku
1. Download Heroku
2. Create a new app
3. Follow Heroku deployment instructions
